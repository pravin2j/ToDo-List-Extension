# ToDo List Chrome Extension

A simple **ToDo List Chrome Extension** that helps you organize tasks directly in your browser.  
It’s lightweight, easy to use, and built using modern web technologies.

## Features

- Add, edit, and delete tasks
- Persistent storage (tasks stay even after browser restart)
- Mark tasks as complete
- Simple UI with minimal design
- Works offline as a browser extension

---

## Built With

- **ReactJS + Vite** — JavaScript Library  
- **Chrome Extension APIs** — Browser integration
